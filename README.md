# MEDSTOCK - Sistema de Equipamentos

Sistema desenvolvido em Python para controle de equipamentos hospitalares.

## 🚀 Funcionalidades
- Registro de equipamentos
- Entrada de mercadorias
- Pesquisa por número de série
- Listagem completa

## 🛠️ Tecnologias
- Python
- Pandas

## ▶️ Como executar
```bash
python medstock.py
